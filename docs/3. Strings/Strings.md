# Strings

Strings in Lesko are surrounded by double quotation marks.

You can display a string literal with the `ecrire()` function:

* Example

```
ecrire("bonjour")
```

### String Length

To get the length of a string, use the `long()` function.

* Example

```
ecrire(long("bonjour"))
```
